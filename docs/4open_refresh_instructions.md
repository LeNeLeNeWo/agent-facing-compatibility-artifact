# 4open Refresh Instructions

1. Commit the updated artifact files to the anonymous GitHub repository used for
   the artifact.
2. Push the changes to that repository.
3. Visit the existing anonymous mirror:
   `https://anonymous.4open.science/r/agent-facing-compatibility-artifact-A3A1`.
4. If 4open does not automatically refresh, regenerate the anonymous mirror from
   the updated GitHub repository URL.
5. Do not put the real GitHub URL into the paper.
6. Keep the anonymous 4open URL in the paper unchanged if the mirror URL remains
   valid; otherwise update the paper in a final patch with the new anonymous
   4open URL only.
