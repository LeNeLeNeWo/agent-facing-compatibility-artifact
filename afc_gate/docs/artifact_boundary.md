# Artifact Boundary

This package is designed for public release. It includes only toy examples and
deterministic mock replay.

Do include:

- Toy trajectories.
- Toy API change specs.
- Unit tests.
- Documentation.
- Deterministic screening rules.

Do not include:

- Provider tokens.
- Private endpoints.
- Raw provider logs.
- Full internal trajectory archives.
- Unredacted user data.
- Experimental artifacts that have not been anonymized.

The paper's real experiment artifacts should be released separately after an
explicit artifact-review pass.
